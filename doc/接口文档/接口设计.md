# 前后端接口设计说明书

## File

On root directory:

* Address:`/api/file/`
* Method:`GET`

* Return: `json`

```
[
	{
		name: "file1",
		size: "1.1MB",
		isDirectory: false,
		modifiedDate: "2019-11-24",
		shareStatus: "私有",
	}
]
```

Other:

* Address:`/api/file/[DirectoryName]/[DirectoryName]/...`
* others are same as above

Directory should be on the top of the list.
